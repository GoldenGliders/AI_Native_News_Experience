# AI_Native_News_Experience

## Overview
**AI Native News Experience** is a web-based platform that delivers personalized, AI-powered news insights. Users can explore dynamic news feeds, generate flashcards for key topics, and interact with an AI assistant to ask questions about any news article.

This project demonstrates the integration of **Spring Boot**, **JavaScript**, and **modern web design** to create a responsive, interactive news portal.

---

## Features

### 1. Personalized Newsroom
- AI-curated news feed tailored to user interests.
- Summarize and simplify complex news stories.

### 2. Flashcards
- Generate flashcards for news topics.
- Displayed in a sidebar for quick reference.
- Interactive and context-aware.

### 3. AI Question Assistant
- Ask questions about any news article.
- AI responds with concise and relevant answers.
- Automatically updates flashcards related to the selected news.

### 4. News Navigator & Video Studio (UI placeholders)
- Planned features for interactive briefings and AI video generation.

### 5. Dark Mode
- Toggle between light and dark themes.
- Persistent across sessions using local storage.

### 6. Responsive Design
- Fully responsive for desktop, laptop, and mobile devices.
- Sidebar for flashcards and question section adjusts based on screen size.

---

## Technologies Used
- **Backend:** Spring Boot (Java)
- **Frontend:** HTML, CSS, JavaScript
- **API Integration:** Gemini AI for AI-powered responses
- **Version Control:** Git & GitHub
- **Deployment Ready:** Configured for cloud deployment (Render, Heroku, etc.)
